# Booleans
The type Boolean represents booleans, and has two values: true and false.

Booleans are boxed if a nullable reference is needed.

Built-in operations on booleans include

`||` – lazy disjunction - Or\
`&&` – lazy conjunction - And\
`!` - negation - Not
